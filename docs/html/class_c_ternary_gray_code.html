Detailed Description

A ternary Gray code generates all strings of n bits in such a way that it is made up of the following bit pairs: 00 01 11, with an additional single bit if n is odd, and each string differs from the previous one in exactly one bit. This class implements a nonrecursive version of the algorithm from the papers

I. Parberry, "A computer assisted optimal depth lower bound for nine-Input sorting networks", Proceedings of Supercomputing '89, pp. 152-161, Reno, Nevada, 1989.

I. Parberry, A computer assisted optimal depth lower bound for nine-Input sorting networks". Mathematical Systems Theory, Vol. 24, pp. 101-116, 1991.
